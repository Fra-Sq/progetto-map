\doxysubsection{Detailed Description}
Observer implementation for handling movement commands within the game. This observer updates the player\textquotesingle{}s current room based on the direction specified in the command.

Processes movement commands (NORTH, SOUTH, EAST, WEST), updating the player\textquotesingle{}s current room accordingly. If the movement is not possible (e.\+g., a wall blocks the way), it returns a message indicating so. Otherwise, it silently updates the player\textquotesingle{}s current room to the new location.
